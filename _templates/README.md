# Templates

Copy these files into the appropriate folder to create new content:

- `essay.md` → copy to `essays/your-essay-name.md`
- `book-review.md` → copy to `book-reviews/your-review-name.md`
- `project.md` → copy to `projects/your-project-name.md`

Then update the front matter (title, date) and write your content.

This folder starts with `_` so Jekyll ignores it — these files won't appear on your site.
